  - text: |
      The [Network Plus in Terahertz Systems Science](https://thznetworkplus.org/) and [AFMNET](https://afmnet.co.uk/) are academic networking platforms enabling researchers to create profiles, connect, and collaborate. Both projects required a clear, accessible structure to support user registration, interaction and ongoing engagement.

  - image:
      - /assets/networks-1.jpg    

  - heading: What I did
    text: |
      - Led end-to-end design and development of both platforms in tandem, from concept through to deployment
      - Defined site architecture, core layouts and interface patterns in collaboration with stakeholders, ensuring clarity and consistency
      - Designed user journeys using interactive prototypes to refine navigation and interaction design
      - Developed a custom WordPress theme, extending core functionality across both the front-end and back-end
      - Built secure forms with both client-side and server-side validation to ensure accessibility, reliability and user experience
      - Streamlined and restructured the WordPress admin interface, removing unnecessary complexity to improve usability
      - Created comprehensive documentation to support administrators and ongoing site management
      - Prioritised accessibility throughout, referring to GOV.UK design standards for forms, search and filtering to ensure an evidence-based, inclusive experience
